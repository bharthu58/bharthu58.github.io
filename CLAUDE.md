Jekyll-based personal blog using Chirpy theme.

## Commands
- Install: bundle install
- Serve: bundle exec jekyll s

## Content
- Posts in _posts/YYYY-MM-DD-title.md

## Rules
- Preserve front matter format
- Maintain Markdown readability
- Do not break Jekyll structure

## Style
- Clear, concise writing
- Avoid unnecessary verbosity

## Workflow
- Edit specific posts/pages only
- Do not refactor entire site unless asked
